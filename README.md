# StockMaster

**StockMaster** est une application Java console pour gérer un stock de produits.  
Elle permet d’ajouter, modifier, supprimer, rechercher des produits et calculer la valeur totale du stock.  
Les données sont sauvegardées automatiquement dans un fichier texte (`stock.txt`) pour conserver les informations entre les sessions.

## Fonctionnalités

- Ajouter un produit : code, nom, quantité, prix  
- Modifier un produit existant  
- Supprimer un produit par son code  
- Afficher la liste complète des produits  
- Rechercher un produit par code  
- Calculer la valeur totale du stock  
- Sauvegarde automatique dans `stock.txt`
